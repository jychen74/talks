# Talks

This repository hosts talk slides and presentation materials for https://jychen74.github.io/talks/

- `index.html` is the public talks list.
- Dated folders such as `20260630/` contain slide decks.
- `abstracts/` contains talk detail pages.
- `assets/images/` contains images used by talk pages.
- The `jychen74.github.io` repository keeps lightweight `_talks` metadata for CV generation.
